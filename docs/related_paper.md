# Related Paper

The mathematical development of Morse-frame persistence is being prepared in a
separate manuscript by Bertrand and Najman.

This public repository is the software repository. It contains the C++ core,
Python interface, examples, tests, documentation, benchmark drivers, and public
benchmark tables. It does not contain the manuscript source, manuscript PDFs, or
private discussion notes.

When a preprint or published version is available, this page will link to it.

For now, the closest public references are:

- Gilles Bertrand, *Morse Sequences: A Simple Approach to Discrete Morse
  Theory*, Journal of Mathematical Imaging and Vision, 2025.
- Gilles Bertrand and Laurent Najman, *Morse Frames*, DGMM 2024.
- Gilles Bertrand and Laurent Najman, *Computing Gradient Vector Fields with
  Morse Sequences*, DGMM 2026.
- Gilles Bertrand, *Morse Sequences on Stacks and Flooding Sequences*, IWCIA
  2026.
